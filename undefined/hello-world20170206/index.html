基于Hexo和coding搭建博客

发表于 2017-02-06   |   分类于 搭建博客   |   阅读次数

###在windows下（linux下步骤也差不多）用coding和hexo搭建自己的博客

####需要的工具：node，git ，以及在coding下建立一个项目

  1. 首先去git！官网下载git，而linux的git安装则比较方便，在终端输入：

    sudo add-apt-repository ppa:git-core/ppa
    sudo apt-get update
    sudo apt-get install git

  2. 安装node：进入node！官网，下载最新版本node，安装

  3. 安装hexo：打开node.js ，输入

    npm install -g hexo

  4. 初始化hexo：在本地目录建一个文件夹，名字随意，比如我d盘下建一个文件夹blog，右键，选择gitbash

    hexo init

  5. 生成静态页面：

    hexo g //生成静态网页
    hexo s //本地调试

然后在浏览器下输入localhost：4000可以看到生成的页面，

  1. 在coding下新建一个项目，登录coding！官网,创建一个新的项目
  2. 建立ssh连接：打开gitbash，输入

    ssh-keygen -t rsa -C

连按三个回车，然后在C:\Users\Administrator目录下会生成一个.ssh文件夹，打开其中的id_rsa.pub，复制全部内容，到coding个人设置，在左下角找到SSH公钥，添加公钥，在公钥内容上粘贴id_rsa.pub的全部内容，公钥名称可不填，添加好以后在gitbash下

ssh -T git@git.coding.net

遇到询问是否信任服务器公钥，输入 yes 即可,如显示hi，XXXXXXX，则表示成功，

  1. 上传静态网页：在coding界面复制git地址，如git@git.coding.net:XXX（此处是你的coding用户名）/XXXX（coding项目名称）.git 然后打开本地hexo初始化目录（blog），在目录下找到_config.yml文件，打开此文件在最下面找到deply

    deploy:
    type: git
    repo: git@git.coding.net:XXXXX/XXXXX.git
    branch: master

改成如上形式，repo后面是你的git地址，记得所有的冒号后面有个空格，branch是你的分支，gitbash

hexo d //上传

等待片刻你的静态网页便上传成功，
